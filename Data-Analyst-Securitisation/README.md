# Data Analyst – Securitisation Risk Analytics

## Overview

This project demonstrates a complete securitisation risk analytics solution developed using Microsoft Power BI and advanced DAX.

The dashboard includes:

- Loan Portfolio Overview
- Investor Reporting Dashboard
- DPD Analysis
- IFRS 9 Expected Credit Loss Analysis
- Vintage Analysis
- Dynamic Loss & Waterfall
- Stress Testing
- Securitisation Risk Arena

## Technologies

- Microsoft Power BI
- DAX
- Power Query
- Data Modeling
- Financial Analytics

## Datasets

- Auto Loan Securitisation Data
- DPD Snapshot History
- Dynamic Loss Monthly
- Static Pool Vintage Data
